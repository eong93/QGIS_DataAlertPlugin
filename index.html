Account Access

Please contact analytics.support@digitalglobe.com for a user account to access the full content and data holdings within DGConnect

  1. Extract the DGConnect folder and move it to the C:\Users.qgis2\python\plugins directory
  2. Open QGIS and go to Manage and Install Plugins
  3. Under Installed, check box next to DGX Plugin
  4. Click the settings icon
  5. Once the username, password, and max items to return have been entered click Validate to ensure everything is working properly
  6. Once all the options have been setup click Save
